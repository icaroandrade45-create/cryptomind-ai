# CryptoMind AI

Plataforma inteligente para análise de criptomoedas e gestão de carteira.

## 🚀 Objetivo

O CryptoMind AI é uma plataforma para investidores em criptomoedas que reúne:

- 📊 Dashboard da carteira
- 💰 Acompanhamento de ativos
- ⭐ CryptoMind Score
- 🤖 Análises com Inteligência Artificial
- 📈 Dados de mercado em tempo real

## Tecnologias

- HTML (atual)
- Supabase
- Vercel
- GitHub
